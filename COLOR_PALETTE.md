# Website Color Palette Reference

This document provides a comprehensive list of colors used throughout the "Our Home Tuition" website to ensure design consistency.

## Primary Theme Colors

| Category | Name | Hex Code | Usage |
| :--- | :--- | :--- | :--- |
| **Primary** | Dark Navy | `#1B2A5A` | Headers, primary buttons, footers. |
| **Secondary** | Terracotta | `#a0522d` | Accents, teaching excellence text. |
| **Accent** | Warm Peach | `#ffb76c` | Highlights, active underlines, counter labels. |
| **Background** | Light Gray | `#F9FAFB` | Page backgrounds. |

## UI Specific Colors

| Category | Name | Hex Code | Usage |
| :--- | :--- | :--- | :--- |
| **Admin Sidebar** | Dark Slate | `#1F2937` | Admin panel sidebar background. |
| **Success** | Emerald | `#10B981` | Positive status badges, success messages. |
| **Error** | Rose | `#EF4444` | Negative status badges, error alerts. |
| **Info** | Sky Blue | `#3B82F6` | Informational notes, CV download buttons. |

## Technical Shades

| Category | Name | Hex Code | Usage |
| :--- | :--- | :--- | :--- |
| **Text (Body)** | Gray 600 | `#4B5563` | Body copy, secondary descriptions. |
| **Text (Title)** | Gray 900 | `#111827` | Main headings, prominent labels. |
| **Border** | Gray 100 | `#F3F4F6` | Card borders, subtle separating lines. |
